# EaseSpace AI Model (ส่วนการเทรนโมเดล)

ซอร์สโค้ดสำหรับกระบวนการเทรนและปรับแต่งความแม่นยำ (Fine-Tuning) ของโมเดลปัญญาประดิษฐ์ (WangchanBERTa) สำหรับทำนายอารมณ์จากข้อความ

## รายละเอียดชุดข้อมูล 
โปรเจกต์นี้ใช้ชุดข้อมูลข้อความภาษาไทยที่รวบรวมและจัดทำขึ้นเพื่อการวิเคราะห์อารมณ์ (Sentiment Analysis) จำนวนทั้งหมด **3,821 ข้อความ** 

**ตัวอย่างข้อมูล**
```text
วันนี้โคตรเฟล ทำอะไรก็ผิดไปหมด | neg
ภูมิใจที่เห็นธุรกิจเล็กๆ ที่เริ่มจากศูนย์ค่อยๆ เติบโตขึ้นด้วยน้ำพักน้ำแรงของตัวเอง | pos
```

หมวดหมู่อารมณ์ (Labels) แบ่งออกเป็น 3 ระดับ ได้แก่
- **neg (Negative)** อารมณ์เชิงลบ / ความเครียด / ความกังวล
- **neu (Neutral)** อารมณ์เป็นกลาง / ข้อความทั่วไป
- **pos (Positive)** อารมณ์เชิงบวก / ความสุข / ความผ่อนคลาย

## สิทธิ์การใช้งาน 
ชุดข้อมูล `dataset.txt` ที่แนบมาในโปรเจกต์นี้ **ไม่มีลิขสิทธิ์** นักพัฒนาและผู้ที่สนใจสามารถนำชุดข้อมูลนี้ไปใช้งาน ศึกษา ดัดแปลง หรือเผยแพร่ต่อยอดในโปรเจกต์อื่นๆ ได้อย่างอิสระโดยไม่ต้องขออนุญาต

## สิ่งที่ต้องเตรียมก่อนเริ่มต้น

1. โปรแกรม **Visual Studio Code**
2. **Python 3.10** (บังคับใช้เวอร์ชันนี้ในการรันคำสั่ง เพื่อป้องกันปัญหาไลบรารีขัดแย้งกัน)
3. **Git** (สำหรับ clone โปรเจกต์)

## ขั้นตอนการตั้งค่าและรันโมเดล

**1. ดาวน์โหลดซอร์สโค้ดโปรเจกต์** เปิด Terminal หรือ Command Prompt แล้วรันคำสั่ง
```cmd
git clone https://github.com/Pathanink/easespace-ai-model.git
```
- จากนั้นเข้าโฟลเดอร์โปรเจกต์
```cmd
cd easespace-ai-model
```

**2. สร้างและเปิดใช้งาน Virtual Environment**
เปิดหน้าต่าง Terminal ใน Visual Studio Code (ตั้งค่าประเภทเป็น Command Prompt) และรันคำสั่งต่อไปนี้เพื่อแยกสภาพแวดล้อม

```cmd
py -3.10 -m venv venv
```
```cmd
venv\Scripts\activate
```
**หมายเหตุ:** เมื่อสำเร็จ จะเห็น (venv) ขึ้นหน้าบรรทัดคำสั่ง

**3. ติดตั้งแพ็กเกจและไลบรารีที่จำเป็น** 
รันคำสั่งเพื่อติดตั้งไลบรารีสำหรับการประมวลผลภาษาธรรมชาติ (NLP) และ Machine Learning

```cmd
pip install -r requirements.txt
```

**4. รันสคริปต์ฝึกสอนโมเดล** ตรวจสอบความเรียบร้อยของไฟล์ชุดข้อมูล จากนั้นเริ่มต้นกระบวนการเทรนโมเดลด้วยคำสั่ง
```cmd
py -3.10 train_model.py
```

## ผลลัพธ์การฝึกสอนและการนำไปใช้งาน
เมื่อการฝึกสอนเสร็จสมบูรณ์ ระบบจะแสดงสรุปผลลัพธ์ความแม่นยำบนหน้าจอ Terminal และสร้างโฟลเดอร์ผลลัพธ์ใหม่ขึ้นมาอัตโนมัติ (เช่น wangchanberta_[วันที่และเวลา]) ซึ่งภายในจะประกอบด้วย

- ไฟล์ประเมินผลการทำงาน: ไฟล์กราฟสถิติรูปแบบ .png (เช่น Learning Curves, Confusion Matrix) และไฟล์ .json สรุปผล

- โฟลเดอร์ final_model/: ประกอบด้วยไฟล์น้ำหนักโมเดล และไฟล์การตั้งค่าที่พร้อมใช้งาน

## การนำโมเดลไปใช้งาน 
- นักพัฒนาสามารถนำโฟลเดอร์ final_model ที่ได้จากการฝึกสอน ไปวางไว้ในโปรเจกต์ฝั่งเว็บไซต์ [easespace-webapp](https://github.com/Pathanink/easespace-webapp) เพื่อใช้งานเป็นโมเดลสำหรับประมวลผลการวิเคราะห์อารมณ์บนหน้าเว็บแอปพลิเคชันต่อไป
